📜 What is a String?

A String is simply a sequence of characters — letters, numbers, symbols — all wrapped inside quotes.

In cybersecurity, almost everything is a string — IP addresses, URLs, usernames, passwords, file paths, HTTP responses. If you master strings, you can manipulate all of that data.

python – string examples 
ip       = "192.168.1.100"     # IP address
url      = "http://target.com/login"  # URL
username = "admin"             # username
path     = "/etc/passwd"       # file path
banner   = 'SSH-2.0-OpenSSH_8.9'  # single quotes work too

print(type(ip))  # <class 'str'>
💡 Single vs Double Quotes: Both "hello" and 'hello' work exactly the same in Python. Pick one style and stick with it!
📏 len() — How Long is a String?

len() tells you exactly how many characters are in a string — including spaces and symbols. This is useful when validating passwords, checking response sizes, or parsing data.

python – string length 
ip       = "192.168.1.1"
password = "P@ssw0rd!23"
empty    = ""

print(len(ip))        # 11
print(len(password))  # 11
print(len(empty))     # 0

# Cybersecurity use: check password length
if len(password) < 8:
    print("❌ Password too short!")
else:
    print("✅ Password length OK")
⬇ output 
11
11
0
✅ Password length OK
🔢 Indexing — Accessing Individual Characters

Every character in a string has a position called an index. Python starts counting from 0, not 1. You can also use negative indexes to count from the end.

Positive indexes →

H0
a1
c2
k3
e4
r5

Negative indexes →

H-6
a-5
c-4
k-3
e-2
r-1
python – indexing 
word = "Hacker"

print(word[0])   # H  — first character
print(word[3])   # k  — fourth character
print(word[-1])  # r  — last character
print(word[-2])  # e  — second from last
⚠️ Common Error: If you try word[10] on a 6-letter word, Python throws an IndexError. Always make sure your index is within range — use len() to check first!
✂️ Slicing — Cutting Out a Piece of a String

Slicing lets you grab a chunk of a string using [start:end]. The start index is included, but the end index is NOT included.

python – slicing 
ip = "192.168.1.1"

print(ip[0:3])   # 192      — characters 0,1,2
print(ip[4:7])   # 168      — characters 4,5,6
print(ip[:3])    # 192      — from start to index 3
print(ip[4:])    # 168.1.1  — from index 4 to end
print(ip[::])    # 192.168.1.1 — full string
print(ip[::-1])  # 1.1.861.291 — reversed! 🤯

# Real use: extract the first octet of an IP
first_octet = ip[:ip.index(".")]
print(first_octet)  # 192

🎯 Memory trick: Think of [start:end] like slicing a loaf of bread — you pick where to start cutting and where to stop. The start slice IS included, the end slice is NOT.
🛠️ String Methods — The Real Power Tools

String methods are built-in functions that you call directly on a string using a dot . — they let you transform, search, and split strings instantly.

Method What it does Cybersecurity use
.upper() Converts to UPPERCASE Normalise data before comparison
.lower() Converts to lowercase Case-insensitive username matching
.strip() Removes spaces from both ends Clean user input before processing
.split() Splits into a list by separator Split IP into octets, parse CSV logs
.replace() Replaces one part with another Sanitise input, modify payloads
.find() Returns index of a substring (-1 if not found) Search for keywords in responses
.startswith() Returns True/False Check if URL starts with "http"
.endswith() Returns True/False Check file extension (.exe, .php)
.count() Counts occurrences of a substring Count failed login attempts in logs
.join() Joins a list into one string Rebuild IP from octets
python – string methods in action 
ip    = "  192.168.1.100  "   # note the spaces
url   = "http://target.com/admin"
fname = "exploit.exe"

# .strip() — clean spaces
clean_ip = ip.strip()
print(clean_ip)            # 192.168.1.100

# .split() — break IP into octets
octets = clean_ip.split(".")
print(octets)              # ['192', '168', '1', '100']

# .startswith() — is it HTTP or HTTPS?
print(url.startswith("https"))  # False — not secure!
print(url.startswith("http"))   # True

# .endswith() — dangerous file extension?
print(fname.endswith(".exe"))   # True — flag it!

# .replace() — mask sensitive data
masked = clean_ip.replace("192.168", "xxx.xxx")
print(masked)               # xxx.xxx.1.100

# .upper() / .lower() — normalise
user = "  ADMIN  "
print(user.strip().lower())   # admin — chaining methods!
⬇ output 
192.168.1.100
['192', '168', '1', '100']
False
True
True
xxx.xxx.1.100
admin
🔗 Method Chaining: You can chain multiple methods in one line!
user.strip().lower() — first strips spaces, then converts to lowercase. Python reads left to right.
🔍 The "in" Keyword — Search Inside a String

The in keyword checks if a substring exists inside another string. It returns True or False. This is one of the most used tools in log analysis.

python – searching with "in" 
log = "Failed password for root from 192.168.1.5 port 22"

# Check if a keyword exists in the log
print("Failed" in log)     # True
print("root" in log)       # True
print("success" in log)    # False

# Real use: alert on suspicious logs
if "Failed" in log:
    print("⚠️  ALERT: Failed login detected!")

# .find() gives you the exact position
print(log.find("root"))     # 22 — found at index 22
print(log.find("admin"))    # -1 — NOT found
⬇ output 
True
True
False
⚠️  ALERT: Failed login detected!
22
-1
🛡️ Mini Project — IP Address Analyser Tool

Let's build a real tool that takes an IP address string and breaks it down completely:

python – ip_analyser.py 
# ===== IP Address Analyser =====

ip = "  192.168.10.55  "   # messy input (spaces)

# Step 1: Clean the input
ip = ip.strip()

# Step 2: Split into octets
octets = ip.split(".")

# Step 3: Analyse
first  = octets[0]
last   = octets[-1]
total  = len(octets)
is_private = ip.startswith("192.168") or ip.startswith("10.")

# Step 4: Display the report
print("=" * 38)
print("     🔍  IP ADDRESS ANALYSER")
print("=" * 38)
print(f"  Full IP       : {ip}")
print(f"  All Octets    : {octets}")
print(f"  First Octet   : {first}")
print(f"  Last Octet    : {last}")
print(f"  Total Octets  : {total}")
print(f"  Private IP?   : {is_private}")
print(f"  Reversed IP   : {ip[::-1]}")
print("=" * 38)
⬇ output 
======================================
     🔍  IP ADDRESS ANALYSER
======================================
  Full IP       : 192.168.10.55
  All Octets    : ['192', '168', '10', '55']
  First Octet   : 192
  Last Octet    : 55
  Total Octets  : 4
  Private IP?   : True
  Reversed IP   : 55.01.861.291
======================================
