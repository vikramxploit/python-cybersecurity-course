🔌 What is a Socket?

A socket is an endpoint for network communication — like a phone. One side dials (connects), the other picks up (listens). Every network connection in computing uses sockets underneath — browsers, SSH, Nmap, everything.

📡 Your First Socket — Single Port Check
python – single port check 
import socket

def check_port(ip, port):
    # Create a TCP socket (AF_INET=IPv4, SOCK_STREAM=TCP)
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

    # Set timeout — don't wait forever
    s.settimeout(1)

    try:
        # Try to connect — returns 0 if open
        result = s.connect_ex((ip, port))
        if result == 0:
            return True   # Port is OPEN
        else:
            return False  # Port is CLOSED
    except Exception:
        return False
    finally:
        s.close()      # always close the socket!

# Test it
ip = "scanme.nmap.org"   # Nmap's legal test server
if check_port(ip, 80):
    print(f"Port 80 is OPEN on {ip}")
else:
    print(f"Port 80 is CLOSED on {ip}")
🎯 connect_ex() vs connect(): Use connect_ex() for port scanning — it returns an error code (0=open) instead of throwing an exception. Much cleaner for scanning loops!
🔍 Banner Grabbing — What Service is Running?

After connecting, some services send a banner — identifying text that reveals the service name and version. This is called banner grabbing and is a key recon technique.

python – banner grabbing 
import socket

def grab_banner(ip, port):
    try:
        s = socket.socket()
        s.settimeout(2)
        s.connect((ip, port))

        # Send a generic HTTP request
        s.send(b"HEAD / HTTP/1.0\r\n\r\n")

        # Receive up to 1024 bytes
        banner = s.recv(1024).decode("utf-8", errors="ignore")
        s.close()
        return banner.strip()
    except Exception:
        return "No banner"

banner = grab_banner("scanme.nmap.org", 80)
print(f"Banner: {banner[:100]}")
⬇ example output 
Banner: HTTP/1.1 200 OK
Server: Apache/2.4.7 (Ubuntu)
Date: Mon, 25 Mar 2026 10:00:00 GMT
🕵️ Why this matters: "Apache/2.4.7" tells you exactly which web server version is running. You can then look up known CVEs (vulnerabilities) for that exact version. This is step 1 of real recon!
🛡️ Mini Project — Full Port Scanner

Putting everything together — a complete port scanner with banner grabbing, service ID, and file output:

python – port_scanner.py (your first real tool!) 
import socket, sys, os
from datetime import datetime

SERVICES = {
    21:"FTP", 22:"SSH", 23:"Telnet",
    25:"SMTP", 53:"DNS", 80:"HTTP",
    110:"POP3", 143:"IMAP", 443:"HTTPS",
    3306:"MySQL", 5432:"PostgreSQL", 8080:"HTTP-Alt"
}

def scan_port(ip, port, timeout=0.5):
    try:
        s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        s.settimeout(timeout)
        result = s.connect_ex((ip, port))
        s.close()
        return result == 0
    except:
        return False

def run_scanner(ip, start, end):
    open_ports = []
    ts = datetime.now().strftime("%Y-%m-%d %H:%M:%S")

    print(f"\n{'='*48}")
    print(f"  🔍 PORT SCANNER  |  Target: {ip}")
    print(f"  Ports: {start}-{end}  |  Time: {ts}")
    print(f"{'='*48}")

    for port in range(start, end + 1):
        print(f"  Checking {port:5}...", end="", flush=True)
        if scan_port(ip, port):
            svc = SERVICES.get(port, "Unknown")
            print(f" ✅ OPEN  → {svc}")
            open_ports.append((port, svc))
        else:
            print(" ❌ closed")

    print(f"\n{'='*48}")
    print(f"  Open ports found: {len(open_ports)}")
    for p, s in open_ports:
        print(f"    → Port {p} ({s})")
    print(f"{'='*48}\n")
    return open_ports

# Scan ports 1-100 on localhost
run_scanner("127.0.0.1", 1, 100)
