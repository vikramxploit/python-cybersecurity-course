🧠 What is a Variable?

Let's understand this with a simple real-life analogy —

Imagine you have a box. That box has a name on it, and something is stored inside it.

A Variable = that box.
Whatever you put inside it — that is the value.

💡 Real Life Example:
target_ip = "192.168.1.1"

This means — we stored an IP address inside a box named target_ip.
Whenever we need that IP, we just write target_ip.
python – creating a variable 
# Creating a variable is super simple
# name = value  — that's all you write

name = "Hacker"         # stored "Hacker" in name
target_ip = "192.168.1.1"  # stored an IP address
port = 8080              # stored a port number

print(name)        # Output: Hacker
print(target_ip)   # Output: 192.168.1.1
print(port)        # Output: 8080
⬇ output 
Hacker
192.168.1.1
8080
🎯 Remember: In Python, you don't write the type when creating a variable — Python figures it out automatically! This is different from Java or C++.
📦 Data Types — 4 Types You Must Know Right Now

Every variable holds a specific type of value. Here are the 4 main types you need to learn right now:

String
Text / Characters — always in quotes
"hello" / "192.168.1.1"
Integer
Whole number — no decimal point
22 / 8080 / 404
Float
Decimal number — has a dot
3.14 / 99.5 / 2.0
Boolean
Only two values — yes or no
True / False
python – see all data types 
# STRING — text always goes inside quotes
hacker_name = "Anonymous"
target_ip   = "10.10.10.5"
os_name     = "Kali Linux"

# INTEGER — whole numbers, no quotes
ssh_port    = 22
http_port   = 80
total_ports = 65535

# FLOAT — decimal numbers
python_ver  = 3.11
risk_score  = 9.8   # CVE severity scores look like this

# BOOLEAN — only True or False (capital T/F)
is_port_open   = True
is_admin       = False
firewall_active = True
⚠️ Common Mistake — Never Do This:

port = "22"  ❌ — This is a String, not a number!
port = 22    ✅ — This is the correct Integer

If you wrap a number in quotes, it becomes text. This will cause errors later when you try to do math with it!
🖨️ print() — Your Very First Best Friend

With print() we can display anything on the screen. It is the most basic and essential function in Python.

python – 3 ways to use print 
ip = "192.168.1.1"
port = 22

# Method 1: print plain text directly
print("Scanning started...")

# Method 2: print a variable
print(ip)

# Method 3: f-string — THE BEST METHOD ✅
print(f"Target IP: {ip} | Port: {port}")
⬇ output 
Scanning started...
192.168.1.1
Target IP: 192.168.1.1 | Port: 22
🌟 What is an f-string?

In an f-string, you can embed variables directly inside curly brackets {}!
It's the cleanest and most professional way to format output — this is what real developers and security engineers use.
🔍 type() — How to Check a Variable's Type?

If you're ever unsure what type a variable is — use type().

python – check the type 
name     = "Hacker"
port     = 22
version  = 3.11
is_open  = True

print(type(name))     # <class 'str'>
print(type(port))     # <class 'int'>
print(type(version))  # <class 'float'>
print(type(is_open))  # <class 'bool'>
