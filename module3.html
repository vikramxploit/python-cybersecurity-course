📋 What is a List?

A list stores multiple values in a single variable — in order. Think of it like a column of boxes, each holding one item. In cybersecurity, lists hold ports to scan, IPs to target, usernames to try, or log lines to analyse.

[0]21
[1]22
[2]80
[3]443
[4]8080
python – creating lists 
# A list of ports to scan
ports = [21, 22, 80, 443, 8080]

# A list of target IPs
targets = ["10.0.0.1", "10.0.0.2", "10.0.0.3"]

# A wordlist for brute force
passwords = ["admin", "password", "123456", "root"]

# Mixed types (possible but rare)
info = ["ssh", 22, True]

print(ports)      # [21, 22, 80, 443, 8080]
print(len(ports))  # 5
print(ports[0])    # 21  — same indexing as strings!
print(ports[-1])   # 8080 — last item
🔧 List Methods — Add, Remove, Sort
python – modifying lists 
ports = [80, 22, 443]

# .append() — add to the end
ports.append(8080)
print(ports)        # [80, 22, 443, 8080]

# .remove() — remove a specific value
ports.remove(22)
print(ports)        # [80, 443, 8080]

# .sort() — sort ascending
ports.sort()
print(ports)        # [80, 443, 8080]

# in — check if value exists
print(80 in ports)   # True
print(22 in ports)   # False — we removed it
🔄 for Loops — Repeat for Every Item

A for loop goes through each item in a list one by one and runs your code for each. This is how you scan every port, try every password, or check every IP — automatically.

python – basic for loop 
ports = [21, 22, 80, 443, 8080]

# Loop through every port
for port in ports:
    print(f"Scanning port {port}...")

print("Scan complete!")
⬇ output 
Scanning port 21...
Scanning port 22...
Scanning port 80...
Scanning port 443...
Scanning port 8080...
Scan complete!
💡 Indentation matters! The code inside a loop must be indented (4 spaces or 1 Tab). Python uses indentation to know what belongs inside the loop. No curly braces like Java/C++!
🔢 range() — Generate Numbers Automatically

range() generates a sequence of numbers without you typing them all out. It's perfect for scanning port ranges.

python – range() examples 
# range(stop) — 0 to stop-1
for i in range(5):
    print(i)    # 0, 1, 2, 3, 4

# range(start, stop)
for port in range(1, 1025):
    print(f"Checking port {port}")  # ports 1 to 1024

# range(start, stop, step)
for port in range(0, 1000, 100):
    print(port)  # 0,100,200,300...900
🎯 Real cybersecurity use: for port in range(1, 65536): — this loops through ALL 65,535 possible ports. That's literally how port scanners work!
🔀 Loop + If — Combine for Smart Logic

The real power comes when you combine loops with if conditions inside — react differently based on each item.

python – loop with conditions 
ports = [21, 22, 80, 443, 3306, 8080]

dangerous = [21, 23, 3306]  # FTP, Telnet, MySQL

for port in ports:
    if port in dangerous:
        print(f"⚠️  Port {port} — HIGH RISK!")
    elif port == 443:
        print(f"🔒 Port {port} — HTTPS (secure)")
    else:
        print(f"✅ Port {port} — OK")
⬇ output 
⚠️  Port 21 — HIGH RISK!
✅ Port 22 — OK
✅ Port 80 — OK
🔒 Port 443 — HTTPS (secure)
⚠️  Port 3306 — HIGH RISK!
✅ Port 8080 — OK
🛡️ Mini Project — Port Risk Analyser
python – port_risk_analyser.py 
# ===== Port Risk Analyser =====

open_ports = [21, 22, 80, 443, 3306, 8080, 23]

services = {
    21: "FTP", 22: "SSH", 23: "Telnet",
    80: "HTTP", 443: "HTTPS",
    3306: "MySQL", 8080: "HTTP-Alt"
}
high_risk = [21, 23, 3306]

risk_ports = []  # collect risky ones

print("=" * 42)
print("     🔍 PORT RISK ANALYSER REPORT")
print("=" * 42)

for port in open_ports:
    svc = services.get(port, "Unknown")
    if port in high_risk:
        print(f"  ⚠️  {port:5} | {svc:10} | HIGH RISK")
        risk_ports.append(port)
    else:
        print(f"  ✅  {port:5} | {svc:10} | safe")

print("=" * 42)
print(f"  Total open  : {len(open_ports)}")
print(f"  High risk   : {len(risk_ports)} ports → {risk_ports}")
print("=" * 42)
